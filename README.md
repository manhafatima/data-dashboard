# WEB102 Project - *Weather Dashboard*

## Overview

This is a React-based Weather Dashboard application that displays real-time weather information for multiple U.S. cities using the WeatherBit API. The dashboard presents current weather conditions in an organized card layout and allows users to explore weather data through search, filtering, and summary statistics.

The purpose of this project is to help users quickly understand weather patterns across different cities through an interactive and easy-to-use dashboard interface.
